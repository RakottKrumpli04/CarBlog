# Verziókezelés
## Helyi repo létrehozása
- inicializálás:
    > git init
- ellenőrzés:
    > git status
- előkészítés commit-re:
    > git add .
- ellenőrzés:
    > git status
- Helyi gépen ellenőzzikük a felhasználónevünket:
    > git config user.name
    
    > git config user.email
- eltároljuk mint egy új verzió:
    > git commit -m "first commit"
